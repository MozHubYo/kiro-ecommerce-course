💡 Kiro 是什麼？

Kiro 是由 Amazon AWS 團隊打造的 AI 開發環境（IDE），基於 VS Code 開源架構，但加入了 AI 自動化的能力。簡單來說，它不只是一個編輯器，更像一個懂你意圖的 AI 工作夥伴 — 你用中文告訴它你想做什麼，它就幫你完成。

📥 安裝步驟

  1. 前往 kiro.dev 官方網站，點擊 Download 按鈕，選擇你的作業系統版本下載安裝檔。
  2. 安裝過程中，把 Other 區塊的選項全部打勾，之後就能通過右鍵啟動 Kiro 。
  3. 第一次開啟 Kiro，會要求你登入帳號。直接用你的 Google 帳號登入就可以了，最快最方便。
