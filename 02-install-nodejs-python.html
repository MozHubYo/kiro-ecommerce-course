🛠️ 環境設置：安裝 Node.js 與 Python

Kiro 背後需要這兩個工具才能幫你跑程式，裝好就不用再管它們了

🟢 安裝 Node.js

前往 nodejs.org，點擊下方的安裝程式，下載後安裝即可。

🐍 安裝 Python

前往 python.org/downloads，點擊 Python 3.14.3 ，下載後安裝即可。

⚠️ 頁面上有兩個下載選項：上面的大按鈕是「Python install manager」，這是 Python 3.14 新推出的版本管理工具，適合需要同時管理多個 Python 版本的開發者。我們只需要裝一個版本就好，所以直接點下面有底線的 Python 3.14.3 下載傳統安裝檔，一路 Next 就能裝完，最簡單。

📖 Node.js 和 Python 是什麼？

Node.js 讓 JavaScript 這個程式語言可以在你的電腦上執行，不只是在瀏覽器裡跑。很多現代的開發工具和自動化腳本都靠它來運作，Kiro 也不例外。

Python 是目前全世界最多人學的程式語言之一，語法接近英文、容易閱讀。它被大量用在資料分析、自動化和 AI 領域，Kiro 在處理這類任務時會需要它。

你不需要學會寫 JavaScript 或 Python，只要裝好讓 Kiro 能呼叫它們就行。
