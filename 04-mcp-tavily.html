🔌 MCP 體驗：用 Tavily 讓 Kiro 上網搜尋

幫 Kiro 裝一個「上網搜尋」的外掛，讓它能查到最新資料

🤔 Kiro 不是已經很聰明了嗎？

Kiro 裡面的 AI 模型確實很強，但它有一個限制：沒辦法上網。它的知識停在訓練資料的截止日期，所以你問它「現在美金匯率多少」或「這個產品最近在亞馬遜賣多少錢」，它答不出來。

要解決這個問題，我們可以幫 Kiro 裝一個叫 MCP 的外掛。MCP 的全名是 Model Context Protocol，你不用記這個名字，只要知道：它是一種讓 AI 工具「接外掛」的標準格式。裝了之後，Kiro 就能用外掛提供的功能，像是上網搜尋、讀取網頁內容等等。

🔍 Tavily 是什麼？

Tavily 是一個專門給 AI 用的搜尋工具。一般的搜尋引擎（像 Google）回傳的是一堆網頁連結，AI 還要自己去讀每個網頁。Tavily 會直接把搜尋結果整理好、重點內容抓出來，AI 拿到就能直接用。

免費帳號每個月有 1,000 次搜尋額度，不用綁信用卡，拿來學習和日常使用很夠。

🔑 第一步：申請 Tavily API Key

API Key 就是一組密碼，讓 Kiro 可以用你的帳號去呼叫 Tavily 的搜尋服務。

  1. 打開 app.tavily.com，點 Sign Up 註冊帳號（可以用 Google 帳號直接登入）。
  2. 登入後會看到你的 API Key。在 OPTIONS 中，點左邊第二個圖示就能複製 API Key
  3. API Key 是你的帳號密碼，不要分享給別人，也不要貼在公開的地方。

⚙️ 第二步：在 Kiro 裡面設定 MCP

  1. 在電腦上新增一個空白資料夾（名字隨便取），對資料夾點滑鼠右鍵，選擇 Open with Kiro。
  2. 看到 Kiro 畫面左邊的導航欄，選擇倒數第二個圖示（Kiro 的 logo）。
  3. 點擊最下方的 MCP SERVERS 區域，再點右上角的 Open MCP Config。
  4. Kiro 會打開一個設定檔，把裡面的內容換成下面這段：
{
  "mcpServers": {
    "tavily": {
      "command": "npx",
      "args": ["-y", "tavily-mcp@latest"],
      "disabled": false,
      "autoApprove": [
        "tavily-search",
        "tavily-extract",
        "tavily_search",
        "tavily_extract",
        "tavily_research"
      ],
      "env": {
        "TAVILY_API_KEY": "貼上你的 API Key"
      }
    }
  }
}
  1. 把 "貼上你的 API Key" 換成你剛才複製的那串 tvly-xxxxxxxxxx，然後存檔（Ctrl + S）。
  2. 關閉Kiro IDE，然後重新開啟，確保設置有正常輸入。
  3. 如果顯示紅色或連不上，先確認 Node.js 有沒有裝好（第二篇有教），然後檢查 API Key 有沒有貼對。

看到 tavily Connected 加上綠色勾勾就代表成功了

🚀 第三步：試試看

設定好之後，直接在 Kiro 右邊的對話區跟 AI 講話，它就會自動用 Tavily 去搜尋。試試看這幾個問題：

  1. 輸入：幫我查一下 2026 年美國對中國商品的最新關稅政策
  2. 輸入：「insulated water bottle」這個品類最近在美國市場有什麼新趨勢？
  3. 輸入：幫我整理 TikTok Shop 美國站最近半年的重要政策變動

你會看到 Kiro 在回答的過程中呼叫了 Tavily 的搜尋工具，回覆裡會包含最新的網路資料和來源連結。

💡 裝了 MCP 之後有什麼不同？

沒有 MCP 裝了 Tavily MCP
查即時資訊 不行，只能用訓練資料回答 可以，直接搜尋最新網路資料
查產品價格 不行，會說「我無法上網」 可以，幫你搜尋並整理結果
查競品資訊 只能給一般性建議 可以搜尋到具體的市場資料
查政策法規 可能過時或不準確 可以查到最新的政策變動

裝了 MCP 之後，AI 就不只能聊天，還能上網查資料，進一步擴展 AI 的能力邊界。
