🔌 讓 Kiro 學會上網搜尋

幫 AI 接上搜尋工具，連結網路查詢最新資料

🧪 先試試看

在 Kiro 右邊的對話區輸入這個問題：

今天西雅圖降雨機率是多少？

它可能會說「我無法上網查詢即時資訊」，或者嘗試用內建的 Fetch 工具（一種上網的指令），拿到相對薄弱的資訊。這就是 AI 普遍遇到的問題：模型被訓練好之後，裡面的資料就凍結了，停在訓練完成的那一天。所以需要接上第三方工具，才能讓模型具備上網的能力。

💡 接上第三方搜尋工具

為了解決聯網的問題，本堂課程我們會使用 Tavily，一個專門給 AI 用的搜尋服務來作演示。

Tavily 免費帳號每月有 1,000 次搜尋額度，而且不用綁信用卡，對日常使用場景來說很夠用了。

市場上類似的工具也很多，大家課後可以自己再做研究，選擇自己偏好的搜尋工具。

⚙️ 設定搜尋工具

  1. 在 Kiro 畫面左邊的導航欄，選擇倒數第二個圖示（Kiro 的 logo）。
  2. 點擊最下方的 MCP SERVERS 區域，再點右上角的 Open MCP Config。
  3. Kiro 會打開一個設定檔。把設定檔中的所有內容刪掉，然後把下方的設定碼完整貼上：
{
  "mcpServers": {
    "tavily": {
      "command": "npx",
      "args": ["-y", "tavily-mcp@latest"],
      "disabled": false,
      "autoApprove": [
        "tavily-search",
        "tavily-extract",
        "tavily_search",
        "tavily_extract",
        "tavily_research"
      ],
      "env": {
        "TAVILY_API_KEY": "tvly-xxxxxxxxxx 每個用戶自己的密鑰"
      }
    }
  }
}
  1. 完成更新後，完整關閉 Kiro 並重新開啟。回到 MCP 介面，檢查 Tavily 後方是否有出現綠色勾勾。
  2. 常見的錯誤包含：代碼貼錯、沒有重新啟動 Kiro，或者 Node.js 沒有安裝好。

🚀 再試一次

再問一次剛才的問題。成功設置的話，會發現 Kiro 呼叫了 Tavily，查詢了更完整的天氣資料。

📋 進階練習：做一份品牌分析報告

請幫我針對「___」這個品牌做一份完整的分析報告。

品牌資訊：
- 品牌名稱：___
- 主營品類：___
- 目前銷售國家：___

請上網搜尋以下資訊，整理成一份結構清楚的報告：

1. 品牌概況：公司背景、創立時間、總部位置、主要產品線
2. 台灣相關新聞：搜尋該品牌近半年在台灣的相關報導或新聞
3. Amazon 表現：在 Amazon 上的重點產品、評價狀況、暢銷排名
4. 市場動態：該品類近期的市場趨勢、競爭對手動向
5. 值得關注的事項：近期的重大事件、政策變動、或潛在風險

完成後把報告存成 report.html，用繁體中文撰寫，排版要清楚好讀。
如果內容較多，請分次寫入檔案。

複製後貼到 Kiro 對話區，AI 會上網搜尋你的品牌資料，整理成一份 HTML 報告。
